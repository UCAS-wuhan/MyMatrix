矩阵类的结构为：
	int  m_nRows; // 矩阵的行数
	int  m_nColumns; // 矩阵的列数
	double* m_lpBuf; // 动态分配用来存放数组的空间
包含的矩阵操作分为操作符重载和功能实现两部分。
一、操作符重载是为了MATLAB中的矩阵赋值和加减等操作习惯能在C++中进行延续。具体包含以下内容：
	1、构造函数、带数值的构造函数、深拷贝构造函数
	2、重写（）和[]括号，可以按照MATLAB方式调用矩阵中具体的值，例如A（2，1）调用第二行第一列的元素，也可以按照C++数组方式取具体数值A[2][1]
	3、重写<<操作符，从键盘读入数值存放到矩阵中,输入时以空格隔开。例如cin<<A;
	4、重写>>操作符，将矩阵输出到屏幕。
	5、重写其他操作符：“=”“+”“-”“+=”“-=”“*”“/”；其中乘法重载为矩阵乘矩阵、单数乘矩阵、矩阵乘单数，除法为矩阵除单数
	6、添加了获取行号函数、获取列号函数、获取矩阵第i行数据的函数、获取矩阵第i列数据的函数。
